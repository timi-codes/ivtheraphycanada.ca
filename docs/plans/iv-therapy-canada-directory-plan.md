## Executive Summary

Canada's public healthcare system is slow and overburdened. Wealthy Canadians are increasingly opting out — paying out of pocket for private, on-demand healthcare services including vitamin IV therapy, mobile IV drips, NAD+ treatments, chelation therapy, and concierge doctor memberships.

The US already has a mature IV therapy industry with chains, franchises, and directories. Canada is 3–5 years behind. There is currently **no dominant national directory** for wellness infusion or concierge medicine in Canada.

**The opportunity:** Build the #1 Canadian directory for wellness infusion therapy — covering IV therapy, vitamin IV drips, NAD+, chelation, and concierge medicine — ranking on Google for `[city] IV therapy`, `[city] vitamin iv drip`, `[city] chelation therapy` searches nationwide. Monetize through vendor listing fees, pay-per-lead, and premium placements.

**Revenue potential:** $10,000–$35,000/month MRR within 12 months.

## Part 1 — Business Plan

### 1.1 Problem & Opportunity

**The problem for consumers:**
- Google searches for "IV therapy Toronto" return a mix of random clinics, no clear directory
- Hard to compare providers, pricing, services, or read consolidated reviews
- No single trusted resource for finding vetted IV therapy or concierge doctors in their city

**The problem for vendors (clinics & mobile nurses):**
- Small operations with no SEO budget or expertise
- Rely entirely on word-of-mouth or Instagram
- No aggregated platform to generate consistent inbound leads
- High margin service ($150–$600/session) but inconsistent booking volume

**The opportunity:**
- First-mover advantage — no dominant Canadian IV therapy directory exists
- Fragmented supplier market = vendors will pay for qualified leads
- High-intent searches with strong commercial value
- Scalable: one platform, 150+ city pages, passive lead generation

---

### 1.2 Business Model

#### Revenue Stream 1 — Monthly Listing Fees (Primary MRR)

| Plan | Price/Month | What's Included |
|------|-------------|-----------------|
| Free | $0 | Basic name, city, phone listing |
| Standard | $149/mo | Full profile, photos, services, priority placement |
| Premium | $299/mo | Featured badge, top of city page, lead notifications, analytics |
| Exclusive | $499/mo | Only vendor shown in that city, all leads go to them |

**Projection:**
- 50 paid vendors × $149 avg = $7,450/mo (Month 6)
- 120 paid vendors × $199 avg = $23,880/mo (Month 12)

#### Revenue Stream 2 — Pay Per Lead

- Charged in cities where no vendor has a paid plan yet
- Lead pricing varies by service type (higher ticket = higher lead fee):

| Service | Lead Fee |
|---------|----------|
| Standard IV therapy / vitamin drip | $20–$40/lead |
| NAD+ therapy | $40–$80/lead |
| Chelation therapy | $80–$150/lead |
| Concierge medicine | $100–$200/lead |

- Lead = someone who filled out "Get a Quote" form
- Billed weekly via Stripe

#### Revenue Stream 3 — Sponsored Content / Blog

- Vendor-sponsored blog posts: $300–$800/post
- "Best IV Therapy in [City]" roundup posts with paid featured placement
- Email newsletter sponsorships as list grows

#### Revenue Stream 4 — Rank & Rent (Long-Term Exit)

- Once high-value city pages rank (Toronto, Vancouver, Calgary), lease or sell them
- City page lease: $500–$2,000/month to exclusive vendor
- Full site acquisition potential: $150K–$500K at 3–5× annual revenue

### 1.3 Target Customer (Vendors)

**Primary vendors:**
- IV therapy clinics (brick & mortar)
- Mobile IV therapy services (nurse comes to you)
- Wellness lounges offering IV & vitamin drips
- Naturopathic clinics with IV and chelation programs
- Integrative medicine clinics (IV + chelation + NAD+)
- Concierge doctors / direct primary care practices
- Med spas with IV therapy and vitamin drip add-ons

**What they look like:**
- 1–5 person operations
- $200K–$2M annual revenue
- Strong Instagram/social but weak Google/SEO presence
- Willing to pay for consistent, qualified bookings
- Already spending $500–$3,000/mo on Google Ads with poor ROI

**Why they'll pay:**
- One new patient from a lead can be worth $400–$2,400/year (recurring visits)
- A $299/mo listing that generates 10 new patients = $4,000+ in revenue
- ROI is obvious and immediate

### 2.2 Database Schema

```sql
-- Vendors (clinics, mobile services, concierge doctors)
CREATE TABLE vendors (
  id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name            TEXT NOT NULL,
  slug            TEXT UNIQUE NOT NULL,
  description     TEXT,
  logo_url        TEXT,
  cover_image_url TEXT,
  phone           TEXT,
  website         TEXT,
  email           TEXT,
  address         TEXT,
  province        TEXT,
  city            TEXT,
  postal_code     TEXT,
  lat             NUMERIC,
  lng             NUMERIC,
  cities_served   TEXT[],
  services        TEXT[],         -- iv_therapy | vitamin_iv | mobile_iv | nad_plus | chelation | concierge | myers_cocktail | glutathione | hangover_iv | immune_iv | hydration
  plan            TEXT DEFAULT 'free',  -- free | standard | premium | exclusive
  stripe_customer_id TEXT,
  stripe_subscription_id TEXT,
  rating          NUMERIC(2,1),
  review_count    INT DEFAULT 0,
  is_verified     BOOLEAN DEFAULT false,
  is_featured     BOOLEAN DEFAULT false,
  created_at      TIMESTAMPTZ DEFAULT now(),
  updated_at      TIMESTAMPTZ DEFAULT now()
);

-- Cities
CREATE TABLE cities (
  id               UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name             TEXT NOT NULL,
  slug             TEXT UNIQUE NOT NULL,
  province         TEXT NOT NULL,
  province_slug    TEXT NOT NULL,
  population       INT,
  lat              NUMERIC,
  lng              NUMERIC,
  nearby_cities    TEXT[],
  meta_title       TEXT,
  meta_description TEXT,
  intro_content    TEXT,          -- unique HTML/MDX per city
  created_at       TIMESTAMPTZ DEFAULT now()
);

-- Leads (quote requests from consumers)
CREATE TABLE leads (
  id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  city            TEXT NOT NULL,
  province        TEXT,
  service_type    TEXT,           -- iv_therapy | vitamin_iv | mobile_iv | concierge | nad_plus | chelation | myers_cocktail | glutathione | other
  name            TEXT NOT NULL,
  email           TEXT NOT NULL,
  phone           TEXT,
  message         TEXT,
  assigned_vendor UUID REFERENCES vendors(id),
  status          TEXT DEFAULT 'new',  -- new | sent | converted | spam
  lead_fee        NUMERIC(10,2),
  billed          BOOLEAN DEFAULT false,
  created_at      TIMESTAMPTZ DEFAULT now()
);

-- Reviews
CREATE TABLE reviews (
  id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  vendor_id   UUID REFERENCES vendors(id) ON DELETE CASCADE,
  author_name TEXT,
  rating      INT CHECK (rating BETWEEN 1 AND 5),
  body        TEXT,
  source      TEXT DEFAULT 'direct',   -- direct | google_import
  created_at  TIMESTAMPTZ DEFAULT now()
);

-- Vendor users (for dashboard access)
CREATE TABLE vendor_users (
  id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  vendor_id   UUID REFERENCES vendors(id) ON DELETE CASCADE,
  user_id     UUID REFERENCES auth.users(id) ON DELETE CASCADE,
  role        TEXT DEFAULT 'owner',
  created_at  TIMESTAMPTZ DEFAULT now()
);
```
